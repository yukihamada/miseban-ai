# ミセバンAI 店舗掲示テンプレート

> **使用方法**
> - A4縦（推奨）またはA3縦で印刷し、店舗入口・カメラ設置箇所の付近に掲示してください
> - `【　】` の箇所を実際の店舗情報に置き換えてください
> - フォントは最低でも12pt（本文）、タイトルは18pt以上を推奨します
> - QRコードはお使いのWebサイトのプライバシーポリシーページに紐付けてください

## 設置に関するガイドライン

### 推奨設置場所

1. **店舗入口の内側または外側** — 入店前に確認できる位置
2. **カメラ設置箇所の付近** — 解析が行われる場所の近く
3. **レジカウンター付近** — お会計時に確認できる位置

### 設置時の注意事項

- 掲示物は視認しやすい高さ（目線の高さ：床から140〜160cm程度）に設置してください
- 照明が当たらず読みにくい場所への設置は避けてください
- 定期的に掲示内容が最新のものかを確認してください（最低年1回）
- 掲示内容に変更が生じた場合は、Webサイトのプライバシーポリシーと同時に更新してください

### 印刷推奨仕様

| 項目 | 推奨値 |
|------|--------|
| 用紙サイズ | A4縦（標準版）/ A3縦（大型店向け） |
| 用紙種類 | 光沢なし（マット）コート紙 |
| 本文フォントサイズ | 12pt以上 |
| タイトルフォントサイズ | 18〜24pt |
| ラミネート加工 | 推奨（屋外・入口付近の場合は必須） |
