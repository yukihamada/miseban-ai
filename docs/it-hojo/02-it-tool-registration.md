## 2. 製品概要

### コンセプト

ミセバンAIは、**既存の防犯カメラやスマートフォンをAI店長に変える**クラウド型店舗分析ソリューション。高価な専用機器の購入不要で、今ある設備の映像をクラウドAIが分析し、来客数カウント・属性推定・ヒートマップ・防犯アラート・AI経営アドバイスをリアルタイムで提供する。

### 技術基盤

| 技術要素 | 詳細 |
|----------|------|
| AI推論エンジン | YOLOv8ベースの物体検出（ONNX Runtime） |
| バックエンド | Rust（高性能・安全性重視） |
| クラウド基盤 | Fly.io（日本リージョン: nrt） |
| データベース | PostgreSQL（Supabase） |
| エッジ処理 | Docker / Raspberry Pi / スマートフォン対応 |
| 通信プロトコル | RTSP / ONVIF / MJPEG / HTTPS |

### セキュリティ設計

- **映像データの保護**: エッジ（Agent）側でフレームキャプチャ・送信。クラウドには分析結果のメタデータのみ保存。生映像はクラウドに保存されない
- **通信暗号化**: 全通信はHTTPS/TLS暗号化
- **認証**: JWT（JSON Web Token）ベースの認証
- **脆弱性管理**: cargo-deny、trivy、gitleaksによる継続的セキュリティ監査
- **コンテナ署名**: cosignによるイメージ署名・検証
- **SBOM**: ソフトウェア部品表の自動生成

## 3. 対応プロセス（補助金申請枠との対応）

### 業務プロセスへの対応

| プロセス | 対応機能 | 対応状況 |
|----------|----------|----------|
| **顧客対応・販売支援** | 来客数カウント、属性推定（年齢・性別）、ピーク時間分析、AI経営アドバイス | 対応 |
| **総務・人事・給与・労務** | シフト最適化提案、人員配置アドバイス（時間帯別来客データに基づく） | 対応 |
| **その他業務固有プロセス** | 防犯アラート（不審行動検知・侵入検知・混雑検知）、店舗動線分析 | 対応 |

### 汎用プロセスへの対応

| プロセス | 対応機能 |
|----------|----------|
| **データ分析・見える化** | ダッシュボード、ヒートマップ、日次レポート、CSV出力 |
| **通知・共有** | LINE通知、Slack通知、リアルタイムアラート |

### 申請枠の適合性

| 申請枠 | プロセス数 | 補助額 | 適合理由 |
|--------|-----------|--------|----------|
| **通常枠（1～3プロセス）** | 3プロセス | 5万～150万円 | 顧客対応・人事・業務固有の3プロセスに対応 |
| **通常枠（4プロセス以上）** | 4プロセス以上 | 150万～450万円 | 上記＋データ分析の汎用プロセスで4以上 |
| **複数者連携枠** | - | 最大3,000万円 | 商店街・商業施設での一括導入に最適 |

## 5. 導入フロー

```
[Phase 1: 準備] 1日
  ├─ 現状ヒアリング（既存カメラの確認、課題把握）
  ├─ 最適プランの提案
  └─ アカウント発行

[Phase 2: 導入] 1日
  ├─ エージェントソフトウェアのインストール
  │   ├─ 方法A: PCにDockerで導入
  │   ├─ 方法B: スマートフォンアプリ
  │   └─ 方法C: Raspberry Pi
  ├─ カメラとの接続設定
  └─ 初期動作確認

[Phase 3: 定着支援] 2週間
  ├─ ダッシュボード操作研修
  ├─ LINE/Slack通知設定
  ├─ レポートの読み方レクチャー
  └─ 14日間無料トライアル期間

[Phase 4: 活用支援] 継続
  ├─ 月次データレビュー
  ├─ AI経営アドバイスの活用指導
  ├─ 機能アップデート案内
  └─ 技術サポート
```

## 7. AI機能の申告

### AI技術の分類

| 項目 | 回答 |
|------|------|
| AI機能の有無 | **あり** |
| AI技術の種別 | **生成AI以外のAI技術** |
| 具体的なAI技術 | コンピュータビジョン（物体検出・人物検出） |
| 使用モデル | YOLOv8n（ONNX形式） |
| AI処理の場所 | エッジ（Agent側）およびクラウド |

### AI機能の詳細

| AI機能 | 技術手法 | 用途 |
|--------|----------|------|
| 人物検出・カウント | YOLOv8 物体検出 | 来客数の自動カウント |
| 属性推定 | 画像分類モデル | 年齢層・性別の推定 |
| 動線分析 | トラッキングアルゴリズム | ヒートマップ生成 |
| 異常検知 | 行動認識モデル | 不審行動・侵入・混雑の検知 |
| 経営アドバイス | 統計分析 + ルールベース | データに基づく改善提案 |
