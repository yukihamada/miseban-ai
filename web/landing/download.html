カメラエージェント ダウンロード

Raspberry Piに5分でインストール。既存カメラをAI分析に接続します。

MisebanAI Camera Agent

Raspberry Pi 4 / 5 (64-bit OS) 対応

ARM64 (aarch64) 静的リンク v0.1.0 ~3 MB
ダウンロード (.tar.gz)

インストール手順

1

Raspberry Piにダウンロード

Pi上でターミナルを開き、以下を実行します。

curl -fsSLO https://misebanai.com/releases/miseban-agent-arm64.tar.gz
tar xzf miseban-agent-arm64.tar.gz
cd miseban-agent
2

インストーラを実行

ffmpegのインストールとsystemdサービスの設定を自動で行います。

./install.sh
3

セットアップウィザードを開く

ブラウザで http://<PiのIPアドレス>:3939 にアクセスします。

# IPアドレスの確認
hostname -I
4

ペアリングコードを入力

ダッシュボードのカメラタブで「ペアリングコードを発行」をクリックし、表示された6桁のコードをウィザードに入力します。

5

カメラを選択して完了

ネットワーク上のカメラが自動検出されます。カメラを選択すると、フレームの送信が開始されます。

動作要件

  • Raspberry Pi 4 Model B / Pi 5 (64-bit Raspberry Pi OS)
  • ffmpeg (インストーラが自動インストール)
  • RTSP対応IPカメラ、またはHTTPスナップショット対応カメラ
  • インターネット接続 (フレームをクラウドAPIに送信)
  • ミセバンAIアカウント (無料登録)

ワンライナーインストール (上級者向け)

$
curl -fsSL https://misebanai.com/releases/miseban-agent-arm64.tar.gz | tar xz && cd miseban-agent && ./install.sh

サービス管理

$
# ステータス確認
sudo systemctl status miseban-agent

# ログ確認
sudo journalctl -u miseban-agent -f

# 再起動
sudo systemctl restart miseban-agent

# 停止
sudo systemctl stop miseban-agent
