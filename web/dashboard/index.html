ミセバンAIへようこそ！

ミセバンAIは、お店のカメラ映像をAIで分析し、
来客数のカウントやピーク時間帯の把握、
異常検知をリアルタイムで行うサービスです。

かんたん3ステップで初期設定を完了しましょう。

店舗情報を入力

AIが最適な分析を行うために、お店の情報を教えてください。

カメラを接続

お使いのカメラのRTSP URLをダッシュボードの「カメラ管理」画面から追加できます。

RTSP URL の形式:
rtsp://[ユーザー名]:[パスワード]@[IPアドレス]:554/stream1

1. カメラの管理画面でRTSP配信を有効にします
2. カメラのIPアドレスとポート番号を確認します
3. ダッシュボードの「カメラ管理」で追加します

セットアップ完了！

ミセバンAIの準備が整いました。
